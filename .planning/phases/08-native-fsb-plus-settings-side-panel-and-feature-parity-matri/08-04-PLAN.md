<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| FSB Agent mode composer -> AgentRuntime::submitTask | User-entered natural language prompt flows to runtime; prompt must not bypass task ownership or leak into diagnostics |
| Tabs mode -> TabModel | Side panel reads tab state but must not mutate tab ownership or routing |
| Tools mode -> page settings | Page toggles affect the active web page; must not bypass permission boundaries |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-04 | Tampering | runTask -> submitTask | mitigate | submitTask is a controlled runtime API; side panel passes prompt + context only; task ID is returned by runtime not caller-set; no secret values in context object |
| T-08-04b | Elevation of Privilege | Tabs mode tab manipulation | mitigate | Tabs mode is display-only (QListWidget); navigation actions use existing m_window->weView()->load() which respects page permissions; no agent tool calls issued from tabs mode |
| T-08-04c | Information Disclosure | Explorer history/bookmark list | mitigate | History and bookmarks are locally-owned data shown only in the native foreground panel; no MCP-accessible read path from this widget |
| T-08-SC | Tampering | npm/pip/cargo installs | accept | No new package manager installs; in-tree C++ Qt code only |
</threat_model>
