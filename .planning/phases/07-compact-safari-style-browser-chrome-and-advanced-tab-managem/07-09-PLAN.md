<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| group metadata -> native chrome | User-created group labels, colors, and collapsed state render in compact chrome and overview/search. |
| tab model roles -> filter/action UI | Model data controls which tabs are visible before user actions are offered. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-09-01 | Information Disclosure | `CompactTabDelegate`, `TabSearchDelegate` | mitigate | Render only group label/color/collapsed metadata plus existing sanitized tab state roles; do not render page text, prompts, secrets, or raw agent params. |
| T-07-09-02 | Tampering | `TabSearchFilterModel` | mitigate | Treat group filters as view filters only; tab mutation remains routed through Plan 07-08 `TabWidget` operations. |
| T-07-09-03 | Denial of Service | Group labels and collapsed sections | mitigate | Bound/elide labels, keep rows fixed at 32px, and expose one representative source row for collapsed groups. |
| T-07-09-04 | Spoofing | Group swatches | mitigate | Use the approved muted swatch list and do not let group colors impersonate automation/supervision accent or destructive health red. |
</threat_model>
