<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| agent command params -> tab state | Untrusted MCP/native command metadata can influence visible tab status. |
| router internals -> UI/MCP reads | Private ownership and supervision maps become summarized UI data. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-02-01 | Information Disclosure | `tabChromeState` | mitigate | Return only owner ID, booleans, health, tool name, and sanitized short reason; never expose params, secrets, vault values, or page content. |
| T-07-02-02 | Tampering | `tabChromeStateChanged` consumers | mitigate | Keep state read-only to UI; destructive action authorization remains in `enforceOwnership`. |
| T-07-02-03 | Spoofing | Agent ownership display | mitigate | Use browser-assigned agent IDs from existing router state, not caller-supplied display text. |
| T-07-02-04 | Denial of Service | Status text rendering | mitigate | Bound and sanitize text before it reaches model roles or UI delegates. |
</threat_model>
