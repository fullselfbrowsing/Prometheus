- Agent dot fallback: 8px with tooltip and accessible name.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| tab model -> compact delegate | Page titles, URLs, and agent state render inside native chrome. |
| user input -> tab operations | Clicks, middle-clicks, context menus, and drags trigger tab actions. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-03-01 | Information Disclosure | `CompactTabDelegate` | mitigate | Render only existing tab title/favicon and sanitized state roles; do not render raw agent params or secrets. |
| T-07-03-02 | Tampering | `CompactTabStrip` drag/drop | mitigate | Delegate movement to `TabModel::dropMimeData` and existing tab move paths. |
| T-07-03-03 | Denial of Service | Delegate text painting | mitigate | Elide titles and bound badge text so long page titles or status strings cannot resize the row. |
| T-07-03-04 | Elevation of Privilege | Context menu actions | mitigate | Use existing `TabContextMenu` and `TabWidget` methods so ownership enforcement additions in later plans can wrap destructive actions centrally. |
</threat_model>
