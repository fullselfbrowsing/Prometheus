<objective>
Vendor the Font Awesome Free 6.x SVG subset and the Poppins/Space Mono brand font TTF files into the falkon source tree, extend icons.qrc with the FA entries, and create fonts.qrc listing the five brand font files.

Purpose: Phase 9 requires every icon and font to be offline at build time. This plan stages all binary assets and their resource manifests so Plans 02-03 can register fonts in C++ and the resolver can load SVGs from :/icons/fa/.
Output: 70 FA SVG files, 5 TTF files, 2 OFL license files, updated icons.qrc, new fonts.qrc.
</objective>
