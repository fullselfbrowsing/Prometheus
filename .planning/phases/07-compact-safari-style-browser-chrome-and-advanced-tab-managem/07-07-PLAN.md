| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-07-01 | Tampering | Session group registry | mitigate | Validate restored group IDs, labels, colors, and collapsed state before applying; drop invalid entries. |
| T-07-07-02 | Denial of Service | Group labels | mitigate | Bound label length and elide in delegates so long names cannot break layout. |
| T-07-07-03 | Information Disclosure | Group persistence | accept | Group labels are user-created local browser metadata; no secrets should be stored in group fields. |

</threat_model>
